# Advanced Classic Clothing [V1.4]
Advanced Classic Clothing is a basic reimplementation of ROBLOX 2D clothing and adds several features which include:
* High resolution textures
* Layerable 2D clothing (with ZIndex support)
* Overlap over the modesty layer for UGC bundled rigs

For a broad list of features and potential uses, view my original post on Devforum

## Documentation

There is no dedicated documentation site for ACC at the moment, For now, the complete documentation is still hosted on the ROBLOX Devforum which also includes examples, sprite mapping tutorials, tips and tricks and method documentation.

The Devforum and `_README_.lua` also includes changelog history before V1.4

## Premade Builds/RBXMs

If you are looking for the `.rbxm` file to include to your existing project, get the model from the Roblox Creator store, assuming it has not been deleted by Roblox again*. Otherwise the current build (alongside previous builds) can be found here, again as a `.rbxm` file.

## Building from source/Contribute
You have to install and configure **Rojo** where you would sync ACC to a Roblox Studio project. It would sync Advanced Classic Clothing in the `ReplicatedStorage` folder. You can use your favorite IDE (such as Visual Studio Code) to make script changes outside of Roblox Studio


> *As Advanced Classic Clothing evolves and more code changes are added, there is a solid chance that ACC would be taken off ROBLOX content distribution again, which is why this repository now exists. I made sure to not use `AssetService` or any services which *may* flag Content guideline moderation. In addition, I was told to host the project here (just in case) because the scope of the project has become larger.
